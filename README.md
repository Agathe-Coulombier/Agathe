## Quick start
```bash
pnpm install   # or: npm i
pnpm dev       # or: npm run dev
```
Visit http://localhost:3000/en or /fr. The middleware redirects based on your browser's language (fallback EN).

## Customize
- Edit copy in `i18n/messages/*.json`
- Update colors & background in `app/globals.css` and Tailwind theme
- Replace links in `Contact.tsx`
- Add real project links and images in `Projects.tsx`
